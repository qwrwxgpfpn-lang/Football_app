# Football 5 A Side
Elite booking and league management platform for the beautiful game.

## Overview
This project is a high-performance, premium web application built with Next.js 15 for managing 5-a-side football matches, bookings, and team stats.

## Getting Started
First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Premium design system)

## Features
- Interactive match booking grid
- Real-time spot availability tracking
- Responsive, athletics-focused UI
- Glassmorphic UI components
